DOSSIER TEXTURES — NEON COMPOUND
================================
Déposez ici vos textures. Noms attendus (modifiables dans
shared/config.js, sections TEXTURES et TEXTURE_REPEAT) :

  sky.jpg        panorama équirectangulaire 2:1 → remplace la skybox du jour
  floor.jpg      sol de l'arène (répétée 10x)
  wall.jpg       murs d'enceinte
  crate.jpg      caisses, murets, piliers, cabanes
  platform.jpg   plateforme centrale
  ramp.jpg       rampes

Formats : .jpg / .png (changez l'extension dans config.js).
Si un fichier est absent, le jeu garde sa couleur unie d'origine.
